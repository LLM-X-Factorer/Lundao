# PPTX 文件上传说明

请将每篇论文的完整 PPTX 文件上传到此目录。

## 文件命名规则

每个 PPTX 文件必须按照论文 ID 命名：

- `daily-0001.pptx` - 论文 1: Hierarchical Reasoning Models
- `daily-0002.pptx` - 论文 2: OpenTSLM
- `daily-0003.pptx` - 论文 3: Delethink
- `daily-0004.pptx` - 论文 4: RLVR
- `daily-0005.pptx` - 论文 5: Multi-Agent Collaborative Reasoning
- `daily-0006.pptx` - 论文 6: Chain of Thought Prompting
- `daily-0007.pptx` - 论文 7: Cultural Understanding in Vision-Language Models
- `daily-0008.pptx` - 论文 8: Hallucination Detection in Financial AI

## 文件要求

- **格式**: PowerPoint 2016+ (.pptx)
- **大小**: < 10 MB (推荐), 最大 < 20 MB
- **内容**: 幻灯片数量应与对应的截图数量一致

## 示例目录结构

```
public/ppt-files/
├── daily-0001.pptx
├── daily-0002.pptx
├── daily-0003.pptx
├── daily-0004.pptx
├── daily-0005.pptx
├── daily-0006.pptx
├── daily-0007.pptx
└── daily-0008.pptx
```

## 注意事项

1. **文件命名必须精确匹配**（区分大小写，全部小写）
2. **不要修改文件扩展名**（必须是 .pptx，不是 .ppt）
3. **确保文件完整性**（上传后测试能否正常打开）

详细说明请参考：`/public/RESOURCE_MAPPING.md`
