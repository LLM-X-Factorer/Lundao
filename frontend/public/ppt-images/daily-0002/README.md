# PPT 截图上传说明

请将本论文的 PPT 幻灯片截图上传到此目录。

## 命名规则
- `slide-1.png` - 第 1 张幻灯片
- `slide-2.png` - 第 2 张幻灯片
- `slide-3.png` - 第 3 张幻灯片
- ... (依此类推，序号连续)

## 图片规格
- **格式**: PNG 或 JPG
- **尺寸**: 1920×1080 (推荐) 或 1280×720
- **大小**: 单张 < 500 KB

## 示例
```
public/ppt-images/daily-XXXX/
├── slide-1.png
├── slide-2.png
├── slide-3.png
├── slide-4.png
└── ...
```

详细说明请参考：`/public/RESOURCE_MAPPING.md`
